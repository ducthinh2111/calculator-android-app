# Calculator Application
A simple calculator application with essential functions like adding, subtracting, multiplying, dividing, modular and square root.
## Technologies:
  - Google API: 27
  - Language: Java
  - IDE: Android Studio
## What I learned:
  - Reverse Polish Notation
  - Android layout
